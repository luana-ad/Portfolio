# Templates e Checklist

Use estes templates para padronizar os READMEs dos projetos.

## Checklist de publicação de um projeto
- [ ] README com objetivo claro
- [ ] Print(s) do dashboard (overview + 1 detalhe)
- [ ] Arquivo do dashboard (pbix / pdf / notebook / app)
- [ ] Dataset amostra (CSV) ou instruções de como reconstruir a fonte
- [ ] Documentação técnica (ETL / queries principais / índices)
- [ ] Licença / restrições de dados (se houver)
